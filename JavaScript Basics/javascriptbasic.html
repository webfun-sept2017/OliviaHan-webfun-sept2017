JavaScript Basics

JavaScript is a cross-platform, object-oriented scripting language. JavaScript is extremely popular for a variety of reasons. it is a small and lightweight language allowing maximum flexibiity for developers to take it in a bunch of different directions. JavaScript lives inside a host environment (a web browser or Nod server), it can be connected to the objects of these environments to provide programmatic control over them.
